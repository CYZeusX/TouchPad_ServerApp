### Touch Pad - Server App
This is the server that sends command signals to control mouse and cursor movements.
You MUST run this app as Admin to let it pass the firewall, and do the magic!  <br> <br>
Download  https://raw.github.com/CYZeusX/TouchPad_ServerApp/main/TouchPad_Server.exe  <br> <br>

#### 1 Finger:
- tap  > Left Click
- drag > Move Cursor
- long press + drag > Drag Item / Select Text

#### 2 Finger:
- tap  > Right Click
- drag > Scroll (up/down/left/right)

#### 3 Finger:
- drag > left:  Ctrl + Shift + Tab <br>
&ensp;&ensp;&ensp;&ensp;&ensp;&ensp; right: Ctrl + Tab
  
#### 4 Finger:
- tap  > Alt + Tab
- drag > Alt + Tab (left/right)
  
#### 5 Finger:
- tap  > Ctrl + T
- drag > Ctrl + W (left/right)
